# 酒店管理系统 java1150

## 项目概述

酒店管理系统是基于前后端分离的架构模式开发，采用 Spring Boot 与 Vue 技术栈，旨在为酒店行业提供一套高效、易用的管理解决方案。

## 技术栈

- 前端技术：Vue、Element-UI
- 后端技术：Spring Boot、MyBatis Plus
- 数据库：MySQL
- 开发语言：Java

## 功能模块

### 系统角色

系统定义了两类角色：管理员与用户。

#### 管理员

- **登录功能**
- **用户管理**：管理用户账户信息
- **订单管理**：
  - 处理酒店预约订单
  - 查看订单列表
- **房间管理**：
  - 房间列表展示
  - 房间详情查询
  - 添加新房间
  - 修改房间信息
- **评论列表**：查看和管理用户评论

#### 用户

- **注册功能**
- **登录功能**
- **查询空余房间**：查询可预约的房间
- **房间预约**：完成房间预订
- **我的订单**：查看和管理个人预约订单

## 运行环境

- 前端：现代浏览器（建议使用 Chrome、Firefox 或 Edge）
- 后端：JDK 1.8 或更高版本，MySQL 5.7 或更高版本
